# **API Endpoints**
---

## **HTML API**
---

### **Root**
  * GET / - loads React web App

## **JSON API**
---

### **Users**
  * POST /api/users

### **Session**
  * POST /api/session
  * DELETE /api/session
  * GET /api/session

### **Channels**
  * GET /api/channels
  * GET /api/channel/:id
  * POST /api/channels
  * PATCH /api/channel/:id
  * DELETE /api/channel/:id

### **Messages**
  * GET /api/messages
  * POST /api/messages
  * DELETE /api/message/:id

### **Direct Message**
  * GET /api/direct_messages
  * GET /api/direct_message/:id
  * POST /api/direct_messages
  * DELETE /api/direct_message/:id
